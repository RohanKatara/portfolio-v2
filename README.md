## ✨ Features

- **Astro 5** static generation with selectively hydrated React islands
- **Lenis** smooth scrolling paired with **GSAP ScrollTrigger** sequences
- **OGL** WebGL shaders for lightweight visual effects
- Editorial-style typography and layout
- Mobile fallback redirects to the legacy 3D coffee portfolio

## 🛠 Tech Stack

- **Framework**: Astro 5
- **UI**: React (islands) · TypeScript
- **Styling**: Tailwind CSS
- **Motion**: GSAP · Lenis
- **WebGL**: OGL

## 🚀 Getting Started

```bash
npm install
npm run dev      # http://localhost:4321
npm run build
npm run preview
```
